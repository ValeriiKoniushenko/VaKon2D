If you have encountered a potential security vulnerability in this project, please report it to us at \href{mailto:Valerii.Koniushenko@gmail.com}{\texttt{ Valerii.\+Koniushenko@gmail.\+com}}. We will work with you to verify the vulnerability and patch it.

When reporting issues, please provide the following information\+:


\begin{DoxyItemize}
\item Component(s) affected
\item A description indicating how to reproduce the issue
\item A summary of the security vulnerability and impact
\end{DoxyItemize}

We request that you contact us via the email address above and give the project contributors a chance to resolve the vulnerability and issue a new release prior to any public exposure; this helps protect the project\textquotesingle{}s users, and provides them with a chance to upgrade and/or update in order to protect their applications.

For sensitive email communications, please use \href{https://github.com/ValeriiKoniushenko/WindowsProfiler-lib/blob/develop/PGP-key.asc}{\texttt{ our PGP key}}.\hypertarget{md__s_e_c_u_r_i_t_y_autotoc_md36}{}\doxysection{\texorpdfstring{Policy}{Policy}}\label{md__s_e_c_u_r_i_t_y_autotoc_md36}
If we verify a reported security vulnerability, our policy is\+:


\begin{DoxyItemize}
\item We will patch the current release branch, as well as the immediate prior minor release branch.
\item After patching the release branches, we will immediately issue new security fix releases for each patched release branch. 
\end{DoxyItemize}
